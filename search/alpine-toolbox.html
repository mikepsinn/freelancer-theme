What is Alpine.js?

Alpine.js is a Vue.js like javascript framework created by Caleb Porzio .
It's similar syntax to Vue.js, but you get to keep your DOM, and sprinkle in behaviour as you see fit.

About Alpine Toolbox

Alpine Toolbox is a project created by and maintained by Amrit Nagi (Also check out Tailwind Toolbox if you are a fan of Tailwind CSS )
